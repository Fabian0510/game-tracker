# Overview
A web app that allows people to upload their photos and track health for boardgames. 

# Must Have
- Web front end
    - Modern web frame work
    - Ability to upload photo
    - Ability to update photo
    - Ability to increase / decrease health
    - Visually attractive
    - Ability to modify user name
    - Ability to arbritrariliy add and remove users
    - Ability to add "shields" that temporarily absorb damage
    - intuitive user interface
- Framework
    - Must be wrapped in a docker container for portability
